# Chat-App

## Summary
> #### description
> This is a simple chat app that allows people to chat online. It is implemented using websockets.
> #### features
> - public chat 
> - private chat (rooms)
> - chat history

## Languages and Tools used
> - frontend - html, css, js
> - backend - flask
